# For Cynthia

A polished static love page for Cynthia, built as a single `index.html` file with optimized photos in `assets/photos`.

## Music

The page includes a soundtrack dock with these requested songs:

- 有點甜
- 沒有如果
- 像晴天像雨天
- DAMN (feat. 6LACK)

Browsers require a user gesture before audio can play. Copyrighted song files are not included; if you add your own licensed MP3 files, place them in a `music` folder using the filenames referenced in `index.html`.
